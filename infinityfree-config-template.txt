# InfinityFree Configuration Template
# Copy and paste these values into your config/env.php file

## IMPORTANT: Replace these placeholders with your actual values

# Your InfinityFree domain (replace with your actual domain)
API_BASE_URL=https://YOUR-DOMAIN.infinityfreeapp.com
FRONTEND_URL=https://YOUR-DOMAIN.infinityfreeapp.com
CORS_ORIGIN=https://YOUR-DOMAIN.infinityfreeapp.com

# Generate a strong 32+ character secret key for JWT
JWT_SECRET=your-unique-32-character-secret-key-change-this-immediately

# Your InfinityFree MySQL database credentials (get from control panel)
DB_HOST=sql200.infinityfree.com
DB_NAME=if0_XXXXXXX_cbt_portal
DB_USER=if0_XXXXXXX
DB_PASS=your-database-password

## Steps to get your database credentials:
1. Login to InfinityFree control panel
2. Go to "MySQL Databases"
3. Create new database named "cbt_portal"
4. Copy the provided credentials above

## Steps to generate JWT secret:
1. Visit: https://www.allkeysgenerator.com/Random/Security-Encryption-Key-Generator.aspx
2. Generate a 256-bit key
3. Copy and use as JWT_SECRET
